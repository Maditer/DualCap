# 双屏截屏助手
一款为双屏掌机开发的截屏工具，可以在副屏底部上滑来实现快捷截屏的功能
### 功能介绍：
支持截主屏、截副屏、截取双屏并拼接的功能，附送一个副屏回桌面功能
